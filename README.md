## Installation

Install using `CDN`

```html
<script src="https://unpkg.com/browse/aquamark/dist/index.iife.js"><script>
```
Install using `npm`

```sh
$ npm install aquamark
```

## Basic Usage

```typescript
const aquamark = new Aquamark()
aquamark.init()
```
